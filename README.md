### 使用Spring4.3.9与Mybatis3.4.4的整合框架，实现了一个功能相对齐全的网上书城的JavaWeb项目。

功能如下：  
1. 书城客户端  

	- 图书浏览
	- 图书分类浏览
	- 图书购买
	- 客户注册/登录/注销
	- 客户订单查询
2. 书城后台管理端

	- 管理员登录/注销
	- 图书类型信息的增删改查
	- 上架图书信息的增删改查(包括图片信息）
